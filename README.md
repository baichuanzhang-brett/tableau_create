# 数据可视化仪表板

一个类似Tableau的数据可视化工具，支持拖拽式创建图表、数据源上传、自定义背景等功能。

## 功能特点

### 1. 数据源上传

- 支持Excel（.xlsx, .xls）和CSV格式的文件上传
- 自动识别字段并判断维度/度量类型
- 字段预览和类型显示

### 2. 图表编辑

- 支持柱状图和折线图
- 多维度选择（可拖拽多个字段到X轴）
- 多度量选择（可拖拽多个字段到Y轴）
- 实时预览功能
- 透明背景设置

### 3. 自定义背景

- 上传自定义背景图片
- 科技感默认背景
- 图表透明度调整

### 4. 数据持久化

- 本地存储图表配置
- 刷新页面后自动恢复
- 保存数据源和字段信息

## 如何使用

1. **打开文件**：在浏览器中打开 `customizable-dashboard.html`
2. **上传数据**：点击"数据源上传"标签，上传Excel或CSV文件
3. **创建图表**：在仪表板页面，点击图表右上角的"编辑"按钮
4. **配置图表**：
   - 从左侧字段列表拖拽字段到X轴和Y轴
   - 选择图表类型（柱状图或折线图）
   - 调整透明度设置
   - 查看实时预览
5. **保存配置**：点击"保存"按钮应用更改
6. **自定义背景**：点击顶部的"更换背景"按钮上传背景图片

## 技术实现

- **前端框架**：React 18
- **数据可视化**：ECharts 5.4
- **文件解析**：xlsx库
- **数据存储**：localStorage
- **样式**：原生CSS

## 注意事项

- 数据存储在浏览器的localStorage中，清除浏览器数据会导致配置丢失
- 建议使用Chrome浏览器获得最佳体验
- 大文件可能会导致性能问题，建议数据量控制在10,000行以内
